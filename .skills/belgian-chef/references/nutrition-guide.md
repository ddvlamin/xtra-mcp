# Recommended Daily Intake (RDI) Guidelines

For an average adult (2000 kcal diet):

| Nutrient | Recommended Daily Intake |
| :--- | :--- |
| **Calories** | 2,000 kcal |
| **Total Fat** | 70g |
| **Saturated Fat** | 20g |
| **Cholesterol** | 300mg |
| **Sodium** | 2,300mg |
| **Total Carbohydrates** | 260g |
| **Dietary Fiber** | 28g |
| **Total Sugars** | 50g |
| **Protein** | 100g |

## Calculation Methodology
When creating recipes, calculate the percentage of RDI for each category based on the totals of the ingredients used.
